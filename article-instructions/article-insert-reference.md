## Article Insert Reference

---

### Running Code in Tinker (Critical — read before writing any insert script)

PsySH (Laravel Tinker) auto-aliases common classes at startup: `Article`, `Category`, `Str`, `DB`, `ArticleStatus`, `User`, and others. Declaring `use` for these again causes a fatal name-collision error.

**Rules for tinker-compatible scripts:**

1. **No `<?php` opening tag.** PsySH evaluates raw PHP expressions, not files. The `<?php` tag causes a parse error when piped via stdin.
2. **No `use` statements.** PsySH auto-aliases the common app classes. Any `use` statement for an already-aliased class is a fatal error.
3. **Use fully qualified names only for non-auto-aliased classes** (e.g. `\Carbon\Carbon::now()`). Common app models and `Str`/`DB` do not need qualification.

**Correct tinker-ready script template:**

```php
$title = 'My Article Title';
$body  = '<p>Body HTML here</p>';

$slug = Str::slug($title);
$i    = 2;
while (Article::where('slug', $slug)->exists()) {
    $slug = "{$slug}-{$i}";
    $i++;
}

$category = Category::firstOrCreate(
    ['slug' => 'my-category'],
    ['name' => 'My Category', 'description' => null]
);

Article::create([
    'author_id'          => 1,
    'category_id'        => $category->id,
    'title'              => $title,
    'slug'               => $slug,
    'excerpt'            => 'Short summary.',
    'body'               => $body,
    'featured_image'     => null,
    'featured_image_alt' => null,
    'tags'               => ['tag1', 'tag2'],
    'status'             => ArticleStatus::Draft->value,
    'read_time_minutes'  => Article::calculateReadTime($body),
    'published_at'       => null,
]);

echo "Done.\n";
```

**How to execute:**

```bash
php artisan tinker --no-interaction < /tmp/insert_article_my_slug.php
```

The file must have **no `<?php`** and **no `use` statements** at the top. The command pipes it directly as raw expressions — this works in one shot.

### Minimal Programmatic Insert

> **Note:** This example shows all fields. When running in tinker, omit `use` statements — see the "Running Code in Tinker" section above.

```php
$title = 'My Article Title';
$body  = '<p>Body HTML here</p>';

Article::create([
    'author_id'          => $user->id,         // required — must be a users.id
    'category_id'        => $category->id,     // nullable — categories.id
    'title'              => $title,
    'slug'               => Str::slug($title), // must be unique, a-z0-9 and hyphens only
    'excerpt'            => 'Short summary…',  // nullable, max 500 chars
    'body'               => $body,             // HTML string (see rich-text-formatting.md)
    'featured_image'     => null,              // nullable URL string
    'featured_image_alt' => null,              // nullable string
    'tags'               => ['guide', 'tips'], // nullable JSON array of strings
    'status'             => ArticleStatus::Published->value,
    'read_time_minutes'  => Article::calculateReadTime($body), // auto from body word count ÷ 200
    'published_at'       => now(),             // see rules below
]);
```

### `published_at` Rules
| status      | `published_at`                      |
|-------------|-------------------------------------|
| `published` | `now()`                             |
| `scheduled` | future `Carbon` datetime            |
| `draft`     | `null`                              |

### `search_vector` Column
Auto-generated by Postgres (stored generated column). **Never set manually.**

### Using the Factory (tests / seeders)

```php
// Published (default)
Article::factory()->create(['author_id' => $user->id]);

// Draft
Article::factory()->draft()->create(['author_id' => $user->id]);

// Scheduled (default +1 day, or pass explicit datetime string)
Article::factory()->scheduled('2026-05-01 09:00:00')->create(['author_id' => $user->id]);

// With a specific category
Article::factory()->create([
    'author_id'   => $user->id,
    'category_id' => $category->id,
]);
```

Factory default: `author_id` → mentor user, `category_id` → new Category via factory.

### Unique Slug Helper (mirrors controller logic)

```php
$base = Str::slug($title);
$slug = $base;
$i    = 2;
while (Article::where('slug', $slug)->exists()) {
    $slug = "{$base}-{$i}";
    $i++;
}
```
