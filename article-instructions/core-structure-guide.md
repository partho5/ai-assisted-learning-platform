# Sentence structuring
- Write in plain, clear English. Avoid slang, idioms, cultural references, and native-speaker expressions. Use simple vocabulary and direct sentence structures that non-native speakers can easily understand.

- Never use an em dash (—). Rewrite the sentence using a comma instead.

- Never use figurative or indirect phrases. If a sentence requires the reader to interpret a meaning behind the words, rewrite it so the meaning is in the words themselves. Say exactly what you mean. Do not use clever comparisons, parallel structures, or abstract expressions when a plain statement works.
